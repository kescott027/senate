# Proposed Stories (External)

This folder contains proposed stories for external repos (e.g., concord-engine).
These are not part of the senate backlog yet and are intended for manual handoff
until automated integration (Keystone registration) is available.

Rules:
- Keep IDs unique with a clear prefix (e.g., PROPOSED-CE-####).
- Do not add these to `.project_management/Backlog.md` until accepted.
- Include clear evidence links and acceptance criteria for handoff.
