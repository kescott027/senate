# BILL-2026-03-15-AM-LAW-005-COVERAGE-GATE

## Summary
Amend LAW-005 to require a 90% coverage target at phase-end (with explicit waiver and rationale when unmet).

## Proposed Amendment

Add to LAW-005: Testing Standards

### New Section: Phase-End Coverage Target
- Phase-end assessment must report test coverage.
- Target coverage is **90%** for core logic.
- If the target is unmet, a waiver must be documented with rationale, scope, and an expiry date.

## Rationale
Ensures test coverage remains a measurable gate for phase readiness while allowing controlled exceptions.

## Impact
May require additional test investment before phase transitions.

## Vote Recommendation
Approve to enforce coverage discipline at phase boundaries.
